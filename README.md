# Nexus

A premium, high-performance social media platform featuring a sleek Deep Obsidian aesthetic and real-time capabilities. 

## Tech Stack
- Next.js (React Framework)
- Tailwind CSS (Styling)
- MongoDB & Mongoose (Database)
- NextAuth (Authentication)
- Cloudinary (Image Processing & Hosting)

## Getting Started Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` with your browser to see the result.

## Deployment Commands

The project is fully optimized and deployment-ready. For production environments (like Render, Vercel, or AWS), use the following commands:

**Build Command**:
```bash
npm run build
```
*This command generates an optimized production build in the `.next` directory.*

**Start Command**:
```bash
npm run start
```
*This command starts the production server.*

**Important**: Ensure all environment variables found in your `.env` file are added to your hosting provider's environment variables configuration.
